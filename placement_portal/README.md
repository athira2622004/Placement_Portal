A Django-based Placement Portal where students can log in, view companies, and apply for jobs.

## Features
- Student login & dashboard
- View company list (role, location, package)
- Apply for jobs
- View applied jobs
- Admin can manage companies

## Tech Stack
- Python
- Django
- HTML, CSS
- SQLite
- Git & GitHub
